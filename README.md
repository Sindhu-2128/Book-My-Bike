# Book-My-Bike
Book My Bike is a PHP-MySQL based online bike rental system developed as a BCA project. It allows users to search available bikes, place bookings, manage profiles, and submit testimonials, while admins manage listings and bookings through a centralized web application.
## Overview

Book My Bike is a web-based bike rental management system developed as part of my BCA academic project. The application allows users to search, check availability, and book bikes online, while providing admin-side management of bookings, users, and testimonials. The system simplifies the bike rental process through an easy-to-use interface and centralized database management.

## Objectives

Provide an online platform for bike rental booking

Allow users to search bikes and check availability

Manage bookings, user profiles, and testimonials

Reduce manual work and improve rental efficiency

## User Features

User registration and login

Search bikes based on availability

Book bikes online

View booking history

Manage user profile

Submit testimonials and feedback

Contact support via contact page

## Admin Features

Manage bike listings

View and manage bookings

Manage users

View testimonials

Control website content pages

## Technologies Used

Frontend: HTML, CSS, JavaScript

Backend: PHP

Database: MySQL

Server: Apache (XAMPP)

Tools: VS Code, phpMyAdmin
## Application Workflow

User registers and logs in

User searches bikes and checks availability

Booking is placed and stored in database

User can view bookings and submit testimonials

Admin monitors and manages data from backend
